# Shane-Assistant (小瑞助手) 🚀

这是一个基于 **STM32** 和 **微信小程序** 的通用蓝牙调试工具。

### 核心功能
- 📈 **波形监控**：实时绘制传感器数据波形，调参可视化。
- ⚙️ **参数调节**：支持 PID 等参数的远程下发与微调。
- 🎮 **虚拟遥控**：内置手柄界面，可直接控制智能车/机器人。
- 💬 **数据对话**：气泡式聊天界面，支持中英文数据收发。

### 目录说明
- `main.c`: STM32 下位机核心逻辑，包含蓝牙协议解析。
- `小程序源码`: 完整的上位机工程，可直接导入微信开发者工具。
